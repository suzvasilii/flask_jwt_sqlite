Описание проекта

Данный проект, написанный на Flask является одним из моих учебных проектов.

В ходе проекта была реализована JWT авторизация для доступа к определенным ресурсам, а также подключена база данных SQLite.

Исходный код данного проекта был поделен на несколько логических модулей для лучшей масштабируемости.
